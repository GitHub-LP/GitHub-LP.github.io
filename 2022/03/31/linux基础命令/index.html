关机 (系统的关机、重启以及登出 )

init 0 关闭系统(2)
telinit 0 关闭系统(3)
init 6 重启
reboot 重启
logout 注销

文件和目录

cd /home 进入 ‘/ home’ 目录’
cd … 返回上一级目录
cd …/… 返回上两级目录
cd 进入个人的主目录
cd 返回上次所在的目录
pwd 显示工作路径
ls 查看目录中的文件
ls -F 查看目录中的文件
ls -l 显示文件和目录的详细资料
ls -a 显示隐藏文件
ls [0-9] 显示包含数字的文件名和目录名
mkdir dir1 创建一个叫做 ‘dir1’ 的目录’
mkdir dir1 dir2 同时创建两个目录
mkdir -p /tmp/dir1/dir2 创建一个目录树
rm -f file1删除一个叫做 ‘file1’ 的文件’
rmdir dir1 删除一个叫做 ‘dir1’ 的目录’
rm -rf dir1 删除一个叫做 ‘dir1’ 的目录并同时删除其内容
rm -rf dir1 dir2 同时删除两个目录及它们的内容
rm -irf dir1强制删除不提醒
mv dir1 new_dir 重命名/移动 一个目录
cp file1 file2 复制一个文件
cp dir/* .复制一个目录下的所有文件到当前工作目录
cp -a /tmp/dir1 . 复制一个目录到当前工作目录
cp -a dir1 dir2 复制一个目录
touch dir1 创建一个叫做 ‘dir1’ 的文件’
touch -p /tmp/dir1/dir2 创建一个文件树

挂载一个文件系统

mount /dev/sdb1 /mnt/disk 挂载一个叫做sdb1的盘 - 确定目录 ‘/mnt/disk’ 已经存在
umount /dev/sdb1 卸载一个叫做sdb1的盘 - 先从挂载点 ‘/mnt/disk’ 退出

磁盘空间

df 显示已经挂载的分区列表
ls 以尺寸大小排列文件和目录

用户和群组

groupadd 123创建一个123用户组
groupdel 123 删除一个123用户组
groupmod -n 456 123 重命名一个用户组 456 新名字 123 老名字
useradd user1 创建一个新用户
userdel -r user1 删除一个用户 ( ‘-r’ 排除主目录)
passwd 修改口令
passwd user1 修改一个用户的口令 (只允许root执行)

权限

chmod ugo+rwx directory1 设置目录的所有人(u)、群组(g)以及其他人(o)以读（r ）、写(w)和执行(x)的权限directory1文件名字
chmod go-rwx directory1 删除群组(g)与其他人(o)对目录的读写执行权限
chown user1 file1 改变一个文件的所有人属性
chown -R user1 directory1 改变一个目录的所有人属性并同时改变改目录下所有文件的属性
chgrp group1 file1 改变文件的群组
chown user1:group1 file1 改变一个文件的所有人和群组属性
chmod 777 disk1 赋予disk1所有权限rwx

打包和压缩文件

tar -cvf 123.tar file1 创建一个非压缩的包 123.tar 包名 file1文件名
tar -xvf 123.tar /tmp 将压缩包释放到 /tmp目录下

RPM 包

rpm -ivh package.rpm 安装一个rpm包 package.rpm安装包名
rpm -U package.rpm 更新一个rpm包但不改变其配置文件
rpm -F package.rpm 更新一个确定已经安装的rpm包
rpm -e package_name.rpm 删除一个rpm包
rpm -qa 显示系统中所有已经安装的rpm
rpm -V package_name 检查文件尺寸、许可、类型、所有者、群组、MD5检查以及最后修改时间

YUM 软件包升级器

yum install package_name 下载并安装一个rpm包
yum update package_name 更新一个rpm包
yum remove package_name 删除一个rpm包
yum list 列出当前系统中安装的所有包
yum clean packages 清理rpm缓存删除下载的包
yum clean all 删除所有缓存的包和头文件
yum makecache 缓存重构

初始化一个文件系统

mkfs /dev/sdb1 在sdb1分区创建一个文件系统

看文件内容

cat file1 从第一个字节开始正向查看文件的内容
tac file1 从最后一行开始反向查看一个文件的内容
more file1 查看一个长文件的内容
head -2 file1 查看一个文件的前两行
tail -3 file1 查看一个文件的最后三行

网络 - （以太网和WIFI无线）

ifconfig eth33 显示一个以太网卡的配置
ifup eth33 启用一个 ‘eth33’ 网络设备
ifdown eth33 禁用一个 ‘eth33’ 网络设备
ifconfig eth33 192.168.1.1 netmask 255.255.255.0 控制IP地址
